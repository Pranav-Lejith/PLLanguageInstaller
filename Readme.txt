
PL Interpreter Installation Guide
Overview
The PL Interpreter is a simple programming language interpreter designed for ease of use and flexibility. It allows users to run scripts written in PL and supports various programming constructs like variables, functions, loops, and more.

Requirements
Windows operating system
Administrative privileges to install the interpreter
Installation Instructions
Download the Distribution: Ensure you have downloaded the ZIP file containing the install.bat, your_interpreter.exe, and this ReadMe.txt.

Extract Files: Extract the contents of the ZIP file to a temporary location on your computer.

Run the Installer:

Right-click on install.bat.
Select Run as administrator.
Follow the on-screen instructions to complete the installation.
Verify Installation:

Open the Command Prompt (Press Windows + R, type cmd, and hit Enter).
Type your_interpreter (replace your_interpreter with the actual name of your executable) and hit Enter.
If the installation was successful, you should see a welcome message from the PL Interpreter.
Usage
To use the PL Interpreter, you can run it directly from the Command Prompt or create a .pl file with your PL code and run it using the interpreter.

Running the Interpreter
You can start the interactive shell by simply typing:

Copy code
your_interpreter shell
Running a PL Script
To run a PL script, use the following command:

arduino
Copy code
your_interpreter run path\to\your\script.pl
Features
Simple Syntax: Easy to write and read.
Variables: Direct variable assignment without the need for a 'let' statement.
Control Structures: Support for if, elif, else, while, and for loops.
Functions: Define and call functions with arguments.
User Input: Get input from users with customizable prompts.
Type Casting: Convert between different data types.
